# Auto-Loan-securitisation-Risk-Dashboard
Power BI dashboard for analyzing auto loan portfolio risk, delinquency trends, and vehicle-wise loan distribution.

## Overview
This Power BI dashboard analyzes auto loan portfolio performance, securitization risk, delinquency trends, and vehicle-wise loan distribution using interactive visualizations.

## Dashboard Insights
- Total Loan Amount Analysis
- Vehicle Type Loan Distribution
- Portfolio Delinquency Monitoring
- DPD Bucket Risk Analysis
- Default Loan Tracking
- Interactive Filtering & Visualization

- Default Loans: 7
